
# landing-page-estetica-glauciani
Criei uma landing page para uma clínica de estética fictícia usando apenas HTML5 & CSS3

## Objetivo
Desenvolver habilidades de desenvolvimento web ao criar uma landing page acessível e responsiva,
além de aplicar conceitos de Git e GitHub para gerenciar e armazenar os arquivos

## Funcionalidades da landing page
- layout responsivo
- Hero section
- Sessão Sobre
- Sessão de Serviços
- Sessão de Depoimentos
- Sessão de Contato
